# Lena Vehicle Data Core — Simulation Engine

## Overview

The Simulation Engine is responsible for replaying vehicle system behavior using deterministic event logs generated by the Lena Data Model.

It allows full reconstruction of system state over time.

---

## Core Principle

> "If the event log exists, the system can be fully reconstructed."
